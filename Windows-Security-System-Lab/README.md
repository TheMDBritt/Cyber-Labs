# Windows Security & System Administration Lab

## Overview

This lab demonstrates hands-on Windows system administration and cybersecurity fundamentals aligned with cybersecurity and systems administration roles. The focus is on **account management, system security, monitoring, reliability, and documentation**â€”skills required to support secure and reliable enterprise environments.

All tasks were performed in a Windows virtual machine to simulate a real operational environment.

---

## Objectives

* Practice Windows user and account lifecycle management
* Verify security controls and endpoint protection
* Monitor system availability, reliability, and performance
* Review logs and system changes for audit and troubleshooting
* Collect evidence and document findings clearly

---

## Lab Scope

This lab covers:

* Local user and privilege management
* Security logging and audit trails
* Endpoint protection verification
* Patch and update awareness
* Service availability checks
* Network availability and exposure review
* System inventory and configuration baseline
* Performance and reliability monitoring
* Change tracking using logs and update history

---

## Tools & Technologies

* Windows 10/11 Virtual Machine
* PowerShell (Administrator)
* Event Viewer
* Windows Defender
* Windows Firewall

---

## Lab Breakdown

### Phase 1: Identity, Users, and Audit

* Verified current user and privilege context
* Audited existing local user accounts
* Created a controlled test user
* Disabled and re-enabled the account to simulate offboarding/onboarding
* Verified account actions in Windows Security logs

**Skills demonstrated:**

* Identity & Access Management (IAM)
* Least privilege principles
* Audit logging and accountability

---

### Phase 2: System Security & Reliability

#### Endpoint Protection

* Verified Microsoft Defender status and real-time protection

#### Patch Awareness

* Reviewed installed Windows updates and recent patches

#### Service Availability

* Checked critical services for logging, updates, and endpoint protection

#### Network Availability & Exposure

* Verified IP configuration and outbound connectivity
* Reviewed listening ports to assess attack surface

#### Firewall Posture

* Reviewed Windows Firewall profiles and default policies

**Skills demonstrated:**

* Endpoint security verification
* System availability monitoring
* Host-based firewall awareness
* Basic network exposure analysis

---

### Phase 3: Inventory, Baseline, and Monitoring

#### System Inventory

* Collected OS and system baseline information
* Reviewed installed software for inventory awareness

#### Performance & Reliability

* Checked system uptime
* Identified top CPU- and memory-consuming processes

#### Change Tracking

* Reviewed recent system updates
* Analyzed recent system event logs for errors and warnings

**Skills demonstrated:**

* Configuration management awareness
* Performance monitoring
* Change and event analysis

---
